# Diet Tracker Backend

## Proje Açıklaması
Diet Tracker, kullanıcıların günlük beslenme alışkanlıklarını takip etmelerine yardımcı olan bir uygulamadır.

## Kurulum
1. Gerekli bağımlılıkları yükleyin:
   ```
   npm install
   ```

2. Çevre değişkenlerini `.env` dosyasında yapılandırın

3. Sunucuyu başlatın:
   ```
   npm run dev
   ```

## Kullanılan Teknolojiler
- Node.js
- Express
- PostgreSQL
- JWT
- Sequelize
